1. 통계적 표본의 대표성 확인
54만 건에 달하는 전체 데이터에서 1,000개의 표본을 무작위로 추출해 비교해 보았습니다. 모집단의 표준편차가 워낙 커서 걱정했지만 중심극한정리에 따라 표본 평균이 모집단과 매우 유사하게 나타나는 것을 확인하며 통계적 이론이 실제 데이터에서 어떻게 작동하는지 체감할 수 있었습니다.

2. 국가별 판매 패턴 분석
주문량이 가장 많은 영국과 독일 간의 평균 결제 금액 차이를 t-검정으로 분석했습니다. p-value가 0.05보다 크게 나와 두 국가 간에 통계적으로 유의미한 차이가 없다는 결론을 얻었습니다.

3. 시각화와 전처리의 중요성
가격을 기준으로 평일과 주말의 분포를 그릴 때 극단적인 이상치들 때문에 처음에는 그래프가 제대로 보이지 않았습니다. 0에서 20달러 사이로 범위를 제한하고 확률 밀도를 적용했고 전처리와 축 설정 하나가 분석의 결과물을 얼마나 바꿀 수 있는지 크게 깨달았습니다.

4. 예측 모델의 한계 
단가, 국가, 날짜 정보를 넣어 판매 수량을 예측하는 회귀 모델을 만들었을 때 결정계수가 0.5% 수준으로 매우 낮게 나왔습니다. 처음엔 당황스러웠지만 사람의 구매 결정은 단순히 가격이나 요일만으로 결정되지 않는다는 당연한 사실임을 다시 한번 느꼈습니다.
로지스틱 회귀 모델은 98%라는 높은 정확도를 기록했지만 정작 중요한 반품 건수는 거의 하나도 맞히지 못하는 현상이 발생했습니다.
이 문제를 해결하기 위해 데이터를 살펴 본 결과 미국 반품률이 약 38%로 유독 높다는 패턴을 발견했습니다. 변수로 추가하고 클래스 가중치를 조절했더니 1%에 불과했던 반품 재현율(Recall)을 45%까지 끌어올릴 수 있었습니다. 전체 정확도는 낮아지더라도 의미 있는 정보를 찾아내는것에 좀 더 초점을 맞췄습니다.
